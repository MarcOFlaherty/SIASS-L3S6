Interpréter les pourcentages

⚠️ Ligne ou Colonne ? La règle d’or !

Le choix du sens des pourcentages détermine ce que vous démontrez : * Les % en ligne décrivent le comportement d’une sous-population (Ex: Parmi les aînés, X% ont des difficultés.) \(\rightarrow\) Permet de comparer les groupes entre eux. * Les % en colonne décrivent la composition d’un phénomène (Ex: Parmi les élèves en difficulté, X% sont des aînés.) \(\rightarrow\) Permet de voir de qui est composé le groupe “en difficulté”.

Dans notre cas : On veut savoir si le fait d’être aîné ou cadet influence la difficulté scolaire. Le rang de naissance est la variable indépendante (la cause présumée), on la met en ligne, et on calcule des pourcentages en ligne.

2. Mesurer l’indépendance

Test du \(\chi^2\) (Khi-deux)

Le test du Khi-deux

Le Khi-deux (\(\chi^2\)) compare le tableau observé (la réalité) avec un tableau théorique (ce qu’on obtiendrait si le rang de naissance n’avait strictement aucune influence sur les résultats scolaires).

  • 🟢 Excel
  • 🔵 Code R

Excel ne fait pas ce test automatiquement. Il faut : 1. Calculer les effectifs théoriques pour chaque case : (Total Ligne * Total Colonne) / Total Général. 2. Utiliser la fonction =TEST.KHIDEUX(Plage_observée; Plage_théorique) pour obtenir la p-value.

tab_brut <- table(df$V0007, df$V0065)

# 1. Le test complet
resultat_test <- chisq.test(tab_brut)
print(resultat_test)

# 2. Afficher les effectifs théoriques
# resultat_test$expected

# 3. Calculer la force du lien (V de Cramer) avec questionr
cramer.v(tab_brut)

P-value et V de Cramer

💡 La P-value (Est-ce significatif ?)

La p-value (ou \(p\)) vous indique la probabilité que la différence observée soit due au simple hasard de l’échantillonnage. * Si \(p < 0.05\) : Le lien est statistiquement significatif (moins de 5% de chances de se tromper). On rejette l’hypothèse d’indépendance. * Si \(p > 0.05\) : On ne peut rien affirmer de solide, les variations relèvent peut-être du hasard.

🎯 Le V de Cramer (Est-ce fort ?)

Avoir une p-value très petite (\(p < 0.001\)) prouve que le lien existe, mais ne dit pas s’il est fort. Le V de Cramer mesure l’intensité du lien entre 0 (nul) et 1 (parfait). * \(< 0.10\) : Lien très faible. * Autour de \(0.20\) : Lien moyen/significatif en sciences sociales. * \(> 0.30\) : Lien fort.

3. Visualisation

Le Mosaic Plot

Les écarts à l’indépendance

Le mosaicplot est le graphique parfait pour visualiser un tableau croisé. L’aire de chaque rectangle est proportionnelle à l’effectif de la case.

  • 🔵 Code R
tab_brut <- table(df$V0007, df$V0065)

# shade = TRUE ajoute des couleurs basées sur les résidus du test du Chi2
# Bleu = Sur-représentation par rapport au hasard
# Rouge = Sous-représentation par rapport au hasard
mosaicplot(tab_brut, 
           las = 3, # Pivote le texte des axes
           shade = TRUE, 
           main = "Rang de naissance et Résultats scolaires",
           xlab = "Rang de naissance", 
           ylab = "Résultats faibles")

Ce qu’il faut regarder

Observez les cases bleues et rouges. Elles vous indiquent exactement où l’indépendance est brisée. Par exemple, si la case (Aîné \(\times\) Bon élève) est bleue, cela signifie que les aînés réussissent statistiquement mieux que si la répartition était liée au hasard.
